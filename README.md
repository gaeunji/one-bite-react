# React Study Notes

제가 수강한 **React 강의** 내용을 실습하고 정리한 코드들을 모아놓은 저장소입니다. 각 섹션별로 주요 개념과 예제를 실습하며 학습한 내용을 기록했습니다.


## 🛠 사용 기술

- React (CRA 기반 또는 Vite 기반)
- JavaScript (ES6+)
- HTML/CSS (필요 시)
- VS Code, Git, GitHub

## 💡 학습 내용 예시
- JSX 문법
- 컴포넌트 생성 및 props 전달
- useState, useEffect 훅 사용법
- 컴포넌트 간 상태 관리
- 이벤트 처리 및 양방향 바인딩
- 조건부 렌더링 및 리스트 렌더링

## 📌 목적
- 강의 수강 내용을 직접 타이핑하며 실습
- 개념 정리 및 복습용 코드 저장
- 프로젝트 구조 및 GitHub 사용 경험 쌓기
